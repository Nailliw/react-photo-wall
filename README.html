Avaliação: Mural de Fotos

Nesta avaliação, você irá criar um mural de fotos:

Perceba que o gif acima também demonstra funcionalidades extras que não fazem parte do critério inicial de aceitação.

Nesta atividade, você estará demonstrando um entendimento básico do seguinte:

  • manutenção de estados em componentes
  • construção de chamadas de API de dentro do componente React
  • Escolha do método de ciclo de vida correto para chamadas de métodos assíncronos
  • iteração de estado para geração dinâmica de elementos DOM

Início

Faça o fork e clone este repositório: https://gitlab.com/kenzie-academy-brasil/se/fe/react/assessment---react-photo-wall

Em seguida, você precisará instalar as dependências.

Antes de programar, você deve se certificar que esta estrutura funciona ao rodar a aplicação:

npm start

Critério de Aceitação

Os critérios de aceitação estão todos dispostos em src/App.jsx nos comentários. Também os reproduzimos abaixo:

Passo 1: Declarar o estado

O objeto estado deve incluir um array de fotos, que serão iteradas mais tarde e exibidas ao usuário em uma grade.

Passo 2: Declarar um método de ciclo de vida

Este método de ciclo de vida deve: ser chamado depois do componente ser renderizado inicialmente, fazer o fetch de um array de fotos, adicionar esse array de fotos ao estado depois de recebido

Passo 3: Certifique-se de que imagens válidas estejam sendo rederizadas

Cada imagem gerada deve incluir: o nome do arquivo da foto, a id da foto, e a URL da foto

Onde "foto" é a foto atual no array de fotos onde está ocorrendo a iteração.

Passo 4: Estilo

Modifique o src/App.css para incluir estilos que façam o seguinte:

  • fazer com que o cabeçalho e o div .collage fiquem centralizados na página
  • garantir que no máximo 4 fotos sejam exibidas por linha

Bônus

Se tiver tempo de sobra (se não tiver outra avaliação para fazer, por exemplo), você pode considerar aprimorar a qualidade de algumas formas:

  • Você deve ter percebido que, no gif, o cabeçalho permanece no topo da página enquanto as fotos estão sendo roladas. Adicione estilos para conseguir fazer isso na sua versão.
  • Veja se você consegue achar um modo de limitar o número de fotos que são adicionadas a, digamos, 10
  • Depois de fazer os itens acima, veja se você consegue adicionar um botão que, ao ser clicado, adiciona as próximas 10 fotos (ou seja, aumenta o limite de 10 para 20, 30, 40, etc.). Você precisará consultar a documentação sobre event handlers e adicionar outro estado ao seu componente para conseguir fazer isso.

Envio

Você precisa enviar a url de uma aplicação implementada. Se você, em vez disso, enviar um link para um repositório (ou seja, apenas do código), você não receberá pontos.
